1. Principais formatações

1.1 Negrito // Destaque:

Nessa frase temos um termo em negrito brabissimo.usando a tag B (Não Semântica) e nessa frase, tenho um termo em destaque usando STRONG (Semântica)

1.2 Itálico // Ênfase

Nessa frase, temos um termo em Itálico usando a tag I (Não Semântica) e nessa frase, temos um termo em ênfase usando a tag EM (Semântica)

1.3 Texto marcado (ou marca texto kkkk)

Podemos criar também um termo marcado utilizando a tag MARK (Semântica)

1.4 Texto maior // Texto menor

Estou criando um texto um pouco maior usando a tag BIG (Não Semântica) e um texto um pouco menor usando a tag SMALL (Semântica)

1.5 Texto deletado // Texto inserido

Podemos marcar um texto que pode ser lido mas que não deve ser considerado igual este texto usando a tag DEL
e também podemos utilizar uma "lacuna" como se fosse uma inserção de texto igual este texto usando a tag INS
(ambas Semânticas)

1.5.1 Texto sublinhado

Este texto também pode ser lido como inserção, porém este é com a tag U (Não Semântica)

1.6 Texto sobrescrito // Texto subscrito

Podemos escrever uma potência da maneira certa com a tag SUP, saindo de "x20 - x8 + 897" para "x20 - x8 + 897"
e escrever elemento da maneira correta também, saindo de "H2O" para "H20"
(ambas Semânticas)

1.7 Trechos de código

Em trechos de código, utilizamos a tag CODE para mostrar ao usuário que se trata de um trecho de código pois as letras ficam mono-espaçadas

1.7.1 Um código de exemplo (sem tag CODE)

num = int(input('Digite um número: ')) if num % 2 == 0: print(f'O número {num} é par') else: pring(f'O número {num} é impar') print('FIM DO PROGRAMA')

1.7.2 Um código de exemplo (com tag CODE)

num = int(input('Digite um número: ')) if num % 2 == 0: print(f'O número {num} é par') else: pring(f'O número {num} é impar') print('FIM DO PROGRAMA')
